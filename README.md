# 18artMonter
十八数藏，价格监控

### 使用
SYNOPSIS
    main.py  \<flags\>

FLAGS
    --phone=PHONE
        Default: ''
    --code=CODE
        Default: ''
    --keyword=KEYWORD
        Default: ''
    --token=TOKEN
        Default: ''
    --price=PRICE
        Default: ''

  Example:
    python3 main.py --phone=18888888888 --code=000000 --keyword=白日鼠白胜 --price=3850 run
